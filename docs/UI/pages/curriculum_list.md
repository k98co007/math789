## 커리큘럼 목록/상세 페이지 세부 항목

## 레이아웃 구조

```
+-------------------------------------------------------------+
|                    상단: 커리큘럼 목록 제목, 검색, 필터        |
+---------------------+---------------------------------------+
|   좌측: 카테고리/필터 |   중앙: 커리큘럼 리스트, 상세, 신청/수강  |
|   및 즐겨찾기        |   커리큘럼별 콘텐츠, 상세 정보            |
+---------------------+---------------------------------------+
|   하단: 추천/관심/피드백 영역                                |
+-------------------------------------------------------------+
```

### 주요 섹션
- 상단: 목록 제목, 검색, 필터
- 좌측: 카테고리, 필터, 즐겨찾기
- 중앙: 커리큘럼 리스트, 상세, 신청/수강, 콘텐츠
- 하단: 추천, 관심, 피드백


- 커리큘럼 목록 테이블/카드
- 검색 및 필터 기능
- 커리큘럼 상세 정보(제목, 설명, 기간, 강사 등)
- 신청/수강 버튼
- 즐겨찾기/관심 등록 버튼
- 커리큘럼별 콘텐츠 목록
