# 선수 학습/연계 수업 페이지 (Classroom Prerequisite)

## 레이아웃 구조

```
+-------------------------------------------------------------+
|                    상단: 선수 학습 안내, 연계 수업 안내       |
+---------------------+---------------------------------------+
|   좌측: 선수 학습 목록 |   중앙: 상세 내용, 진도, 추천 수업     |
|   및 진입 버튼        |   연계 수업 바로가기, 진도 표시         |
+---------------------+---------------------------------------+
|   하단: 완료/진도/피드백 영역                               |
+-------------------------------------------------------------+
```

### 주요 섹션
- 상단: 안내 문구, 연계 수업 정보
- 좌측: 선수 학습 목록, 진입 버튼
- 중앙: 상세 내용, 진도, 추천 수업, 바로가기
- 하단: 완료, 진도, 피드백


- 선수 학습 항목 안내 및 진입 UI
- 연계 수업 바로가기, 추천 수업 목록
- 선수 학습 진도, 완료 표시
