# 변형/추가 문제 페이지 (Classroom Variant)

## 레이아웃 구조

```
+-------------------------------------------------------------+
|                    상단: 변형/추가 문제 제목, 난이도           |
+---------------------+---------------------------------------+
|   좌측: 문제 목록    |   중앙: 문제 내용, 풀이 입력/해설         |
|   및 선택           |   풀이 결과, 피드백, 추가 요청             |
+---------------------+---------------------------------------+
|   하단: 제출/피드백/다음 문제 이동 영역                      |
+-------------------------------------------------------------+
```

### 주요 섹션
- 상단: 문제 제목, 난이도
- 좌측: 문제 목록, 선택
- 중앙: 문제 내용, 풀이 입력, 결과, 해설, 추가 요청
- 하단: 제출, 피드백, 다음 문제 이동


- 변형 문제, 난이도별 문제 목록
- 문제 풀이 및 피드백 UI
- 문제별 해설, 추가 문제 요청
