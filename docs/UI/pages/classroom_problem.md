# 문제풀이/코딩 페이지 (Classroom Problem)

## 레이아웃 구조

```
+-------------------------------------------------------------+
|                    상단: 문제 제목, 난이도, 타이머            |
+---------------------+---------------------------------------+
|   좌측: 문제 목록    |   중앙: 문제 내용, 풀이 입력/코딩 영역   |
|   및 선택           |   풀이 결과, 해설, 점수, 피드백          |
+---------------------+---------------------------------------+
|   하단: 제출/피드백/다음 문제 이동 영역                      |
+-------------------------------------------------------------+
```

### 주요 섹션
- 상단: 문제 제목, 난이도, 타이머
- 좌측: 문제 목록, 선택
- 중앙: 문제 내용, 풀이 입력, 결과, 해설, 점수
- 하단: 제출, 피드백, 다음 문제 이동


- 수학 문제 목록 및 선택 UI
- 파이썬(Playground) 코딩 인터페이스
- 실시간 채점 및 피드백 영역
- 변형/추가 문제 진입 버튼
- 풀이 결과, 점수, 해설 표시
